# LightMap

A lightweight minimap and world map mod for Minecraft.

## Features

- **Minimap**: Always know where you are with a detailed minimap
- **World Map**: Explore the full scope of your world with an interactive world map
- **Customizable**: Adjust size, position, and zoom level

## Requirements

- **Minecraft Version**: 1.20.1
- **Forge**: Version 47+

## Installation

1. Install Minecraft Forge for 1.20.1
2. Download LightMap
3. Place the `.jar` file in your Minecraft `mods` folder
4. Launch Minecraft

## Keybindings

| Key | Action |
|-----|--------|
| M | Open LightMap Menu |
| Z | Cycle Zoom Level |
| X | Toggle Fullscreen Map |

## Configuration

Settings are saved in `config/lightmap.properties`
